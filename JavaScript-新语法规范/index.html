let 和 const

关于 let 前面讲过了挺多，这里做以下总结：

  • let 声明的变量具有块级作用域的概念；
  • let 声明的变量没有变量提升的概念；
  • let 声明的变量名不能重复；
  • let 声明的全局变量不能用全局对象的属性来访问。

const 除了具备上述 let 的特点之外，const 用于定义常量，这是 const 独有的特点。

Tips：另外就是 let 和 var 均可以先声明后赋值，但 const 不行。
